# Metric Definition

> Purpose · Grain · Refresh for each metric.

| Metric | Purpose | Grain | Refresh | Source Mart |
|---|---|---|---|---|
| Users | unique visitor count | daily | daily | dashboard_kpi |
| Sessions | visit frequency | daily | daily | dashboard_kpi |
| Page Views | content consumption | daily | daily | dashboard_kpi |
| Engagement Rate | quality of visit | daily | daily | dashboard_kpi |
| Scroll Rate | landing page interest | daily · page | daily | landing_page_mart |
| Avg Engagement Time | depth of interest | daily · page | daily | landing_page_mart |
| Channel Sessions | traffic source mix | daily · channel | daily | marketing_channel_mart |
| Funnel Drop-off | conversion bottleneck | daily · step | daily | funnel_mart |
| Journey Path | navigation flow | daily · path | daily | journey_mart |
| Cohort Retention | user loyalty | weekly · cohort | weekly | cohort_mart |
| Returning Users | repeat visit rate | weekly | weekly | cohort_mart |
